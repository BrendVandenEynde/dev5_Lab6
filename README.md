# Live Stats Web App

This web application provides live updates for team statistics. It includes a form to update scores and a live stats view that dynamically updates in all connected browsers.

## Features

- **Update Stats:**
  - Visit `/updatestats` to input scores for different teams.

- **Live Stats View:**
  - Navigate to `/` or `/stats` to see live statistics for all teams.
  - Updates occur in real-time across multiple browser windows.
